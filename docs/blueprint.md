# **App Name**: Scoreboard Saga

## Core Features:

- Player Registration: Capture player details: name, surname, email, and phone number.
- Leaderboard Display: Display a real-time scoreboard showing player names and scores.
- Dynamic Ranking: Automatically update the scoreboard when a new player is added or a score changes, ensuring the list is always ordered by score.

## Style Guidelines:

- Primary color: Dark Blue (#3F51B5) for a professional and focused feel, suitable for a competitive environment.
- Background color: Very light gray (#F5F5F5), nearly white, to provide a clean backdrop that emphasizes readability.
- Accent color: A muted Purple color (#9575CD), a vivid accent that stands out while still aligning with the theme.
- Body and headline font: 'Inter', a sans-serif font for clear and modern text, ensuring easy readability on the scoreboard.
- Use simple, flat icons for player profiles and scoring actions. The icons should be easily recognizable and aligned with the overall minimalistic design.
- A clean, structured layout that focuses on readability and ease of use. The scoreboard should be prominently displayed with clear labels and an intuitive flow.
- Subtle animations for score updates to provide visual feedback without being distracting. Transitions should be smooth and enhance the user experience.
