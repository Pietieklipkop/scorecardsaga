# Scoreboard Saga: Features and User Stories

This document outlines the core features of the Scoreboard Saga application and the corresponding user stories from the perspective of a "Game Master" or "Admin".

---

## 1. Authentication

Secure access to the application is crucial for managing scoreboards.

-   **Feature**: User Registration
    -   **User Story**: As a Game Master, I want to sign up for a new account using my email and a password so that I can create and manage my own private scoreboards.

-   **Feature**: User Login
    -   **User Story**: As a Game Master, I want to log in to my account so that I can access my scoreboards and manage the game.

-   **Feature**: User Logout
    -   **User Story**: As a Game Master, I want to log out of my account to ensure my session is secure and prevent unauthorized access.

---

## 2. Player Management

The Game Master needs to be able to manage the participants of the competition.

-   **Feature**: Add a New Player
    -   **User Story**: As a Game Master, I want to add a new player by providing their name, surname, email, and phone number so they can be included in the competition and appear on the leaderboard.

-   **Feature**: View Player List
    -   **User Story**: As a Game Master, I want to see a list of all registered players so that I can have a clear overview of all participants.

---

## 3. Score and Leaderboard Management

The core of the application is the real-time management of scores and the display of the leaderboard.

-   **Feature**: Update Player Score
    -   **User Story**: As a Game Master, I want to update a player's score at any time so that the leaderboard accurately reflects their current standing in the game.

-   **Feature**: Real-time Leaderboard
    -   **User Story**: As a Game Master, I want to view a real-time leaderboard that is automatically sorted by the highest score, so I and any spectators can instantly see who is in the lead.

-   **Feature**: Activity Log
    -   **User Story**: As a Game Master, I want to see a log of all recent score updates and player additions so that I can track the history of the game and correct any mistakes.

---

## 4. Notifications

Keeping players informed is an important part of the game experience.

-   **Feature**: WhatsApp Notifications
    -   **User Story**: As a Game Master, I want to send a WhatsApp notification to a player when their score is updated so they are immediately informed of the change, even if they are not looking at the scoreboard.
